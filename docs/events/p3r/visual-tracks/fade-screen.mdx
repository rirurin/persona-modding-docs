# Fade Screen
Fade the screen in or out.

import commonStyle from "../common.module.css";
import { useState } from 'react';

:::note
Fade Screen is a special track since it’s not bound to a target object, so it’s stored at the root of the sequence:
![](/assets/images/events/p3r/UE4Editor_D8aL1nbYCp.png)
:::

It’s available in the green new track button under Atlus Event Fade Screen Track. The color parameters determine what the fill color is that the fade is going from or fading into.

## Fade Screen Type
- `EVT_FADESCREEN_FADE_IN`
- `EVT_FADESCREEN_FADE_OUT`
- `EVT_FADESCREEN_CROSSFADE_IN`
- `EVT_FADESCREEN_CROSSFADE_OUT`

A fade ID of -1 will inherit whatever fade type was set previously.
